# 🎵 Acordes Brasil – Chorverwaltung

![License: Proprietary](https://img.shields.io/badge/license-proprietary-red)
![PWA Ready](https://img.shields.io/badge/PWA-ready-brightgreen.svg)

**Brasilianischer Chor in Frankfurt am Main** – Verwaltungs- und Kommunikationsplattform für Chor, Mitglieder und Vorstand.

## Funktionen

### Öffentlicher Bereich
- Mehrsprachig (DE / EN / PT)
- Konzertübersicht
- Informationen zur Mitgliedschaft
- Sponsoring- und Partnerbereich

### Mitgliederbereich
- Login für Mitglieder
- Kalender mit Teilnahme-Funktion
- Push-Benachrichtigungen
- Dokumente, Noten und Audio-Dateien
- Interne Kommunikation

### Vorstandsbereich
- Mitgliederverwaltung
- Event- und Probenverwaltung
- Beiträge, Spenden und Finanzen
- Projekte, Partner und Auswertungen
